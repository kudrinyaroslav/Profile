общее описание


суть работы с приложением:
клиент заходит на сайт http://localhost:8000
клиент заполняет анкету
анкета сохраняется в базе данных


технологии:
python,
django,
postgres,
nginx,
gunicorn

процесс работы приложения
пользователь заходит на главную страницу сайта - выдает nginx статику, динамику и данные из бд спрашивает у gunicorn - который спрашивает у django
вводит свои данные
нажимает кнопку отправить
данные отправляются в базу данных



сервера прилож
web-server
nginx будет в контейнере работать как веб-сервер, конфигурационный файл будет на хосте - динамические файлы будет отдавать gunicorn из django

front-server
динамика и страницы  сайта будут лежать в контейнере front  

база данных
postgres будет в контейнере, конф файл и база лежат на хосте



файловый состав Django:
main.html - единственная страница
